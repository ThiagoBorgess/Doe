# Doe

Doe é um projeto feito na semana Maratona Dev 3.0 disponibilizada pela Rockeseat. Ele visa captar contatos de pessoas para doação de sangue.

## Tecnologias

- Front-End: HTML,CSS e Javascript
- Back-End: NodeJS
